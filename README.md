# 🛠️ ShramSaathi - Empowering Blue-Collar Workers

ShramSaathi is a full-stack web application that helps **blue-collar workers** discover nearby jobs based on **district** and **work type**. It connects workers with potential job opportunities in their area, making the informal job market more accessible and efficient.

## 🔍 Features

- 🔎 **Search Nearby Jobs** by district and type of work
- 👷 **Register Blue-Collar Workers** with essential skillsets
- 🧑‍💼 **Post Jobs** via employer/admin side
- 💡 Real-world use case targeting India’s unorganized labor sector
- 🧑‍🎨 Beautiful custom UI with clean and responsive layout

---

## 🖥️ Tech Stack

| Layer        | Technology                |
|--------------|----------------------------|
| Frontend     | React.js, Axios, Custom CSS |
| Backend      | Spring Boot, Java           |
| Database     | H2 (in-memory, swappable to MySQL) |
